# 縦型時刻表システム 仕様書

## 概要

複数路線の列車時刻表を縦型テーブルで表示し、ダイヤグラム（時空間図）でも可視化できるWebアプリケーション。  
ローカル環境で `serve.bat` を実行して使用する。

---

## ファイル構成

```
jikokuhyou/
├── index.html          # 時刻表ページ
├── diagram.html        # ダイヤグラムページ
├── stations.csv        # 駅マスタ（全路線共通）
├── connections.csv     # 直通運転接続定義
├── serve.bat           # ローカルサーバー起動スクリプト
└── data/
    ├── {路線名}.csv    # 列車時刻データ（路線ごと）
    └── ...
```

**起動方法:** `serve.bat` を実行 → `http://localhost:8080/index.html`

---

## データフォーマット

### 1. stations.csv — 駅マスタ

```
路線名,方向,順序番号,駅名,距離(km)
```

| フィールド | 説明 |
|-----------|------|
| 路線名 | 路線の名称。`data/{路線名}.csv` のファイル名と一致させる |
| 方向 | `上り` / `下り` |
| 順序番号 | 表示順（整数）。小さい方が先頭 |
| 駅名 | 駅の名称 |
| 距離(km) | 先頭駅からの距離。ダイヤグラムのY軸スケールに使用（省略可） |

**例:**
```csv
京急大師線,下り,1,京急川崎,0
京急大師線,下り,2,港町,1.2
京急大師線,上り,1,小島新田,0
```

- 上り・下りは**それぞれ独立して定義**する（駅順が逆になるため）
- 距離が設定されている場合はダイヤグラムのY軸を実距離比で描画

---

### 2. data/{路線名}.csv — 列車時刻データ

```
路線名,方向,列車番号,種別,終着駅,始発時刻,運行日,駅名,発着,時刻
```

| フィールド | 説明 |
|-----------|------|
| 路線名 | `stations.csv` の路線名と一致 |
| 方向 | `上り` / `下り` |
| 列車番号 | 列車の識別番号（文字列可。例: `1234`, `583X`, `A567`） |
| 種別 | `普通`, `急行`, `特急` など |
| 終着駅 | **直通時は実際の最終目的地**を記載（自路線の終点でなくてもよい） |
| 始発時刻 | 自路線での最初の発車時刻（HHMM形式。例: `0730`） |
| 運行日 | 運転条件（後述） |
| 駅名 | その行の対象駅 |
| 発着 | `発` / `着` |
| 時刻 | 発着時刻（HHMM形式）または `レ`（通過） |

**1列車のデータ例（7駅 × 1列車 = 7行）:**
```csv
京急大師線,上り,583X,普通,京急川崎,507,休日,小島新田,発,507
京急大師線,上り,583X,普通,京急川崎,507,休日,大師橋,発,508
京急大師線,上り,583X,普通,京急川崎,507,休日,東門前,発,510
京急大師線,上り,583X,普通,京急川崎,507,休日,川崎大師,発,511
京急大師線,上り,583X,普通,京急川崎,507,休日,鈴木町,発,513
京急大師線,上り,583X,普通,京急川崎,507,休日,港町,発,514
京急大師線,上り,583X,普通,京急川崎,507,休日,京急川崎,着,517
```

**通過駅:**
```csv
東海道・山陽新幹線,下り,1234,のぞみ,博多,600,平日,新富士,発,レ
```

### 3. 運行日フィールドの書式

複数条件を `|`（パイプ）で区切る。`!` を前置すると除外条件。

| 値 | 意味 |
|----|------|
| `毎日` | 毎日運転 |
| `平日` | 月〜金（土日祝除く） |
| `休日` | 土日 |
| `土休日` | 土日（`休日` と同義） |
| `土曜` | 土曜のみ |
| `日曜` | 日曜のみ |
| `M/D` | 特定日（例: `3/15`） |
| `M/D-M/D` | 期間（例: `12/29-1/3`）。年跨ぎ対応 |
| `YYYY/M/D` | 年付き特定日 |
| `!条件` | 除外（例: `平日|!12/29-1/3` = 12/29〜1/3を除く平日） |

**例:**
```
平日              → 平日のみ
休日              → 土日のみ
平日|!12/29-1/3   → 年末年始を除く平日
3/15              → 3月15日のみ
```

### 4. connections.csv — 直通運転接続定義

列車番号が路線境界で変わる直通運転を紐付けるファイル。

```
路線1,方向1,列車番号1,運行日,接続駅,路線2,方向2,列車番号2
```

| フィールド | 説明 |
|-----------|------|
| 路線1 | 接続元の路線名 |
| 方向1 | 接続元の方向 |
| 列車番号1 | 接続元の列車番号 |
| 運行日 | 運行日条件（列車データと同じ書式） |
| 接続駅 | 路線境界となる駅名 |
| 路線2 | 接続先の路線名 |
| 方向2 | 接続先の方向 |
| 列車番号2 | 接続先の列車番号 |

**例（東武東上線〜みなとみらい線 直通）:**
```csv
東武東上線,下り,1234,平日,和光市,東京メトロ副都心線,下り,A567
東京メトロ副都心線,下り,A567,平日,渋谷,東急東横線,下り,A567
東急東横線,下り,A567,平日,横浜,みなとみらい線,下り,A567
```

接続はチェーン状に複数段定義できる（最大10段）。

## 機能仕様

### 時刻表ページ（index.html）

#### フィルタ機能
- **路線選択**: `stations.csv` に登録されている路線を一覧から選択
- **方向選択**: 上り / 下り
- **ダイヤ種別**: 全て / 平日 / 休日
- **日付指定**: カレンダーで特定日を指定 → その日に運転する列車のみ表示

#### 時刻表表示
- 縦型テーブル（駅 = 行、列車 = 列）
- 着・発が両方ある駅は2行に分けて表示
- 通過駅は `レ` を表示
- 列車ヘッダーには列車番号・種別・始発駅・始発時刻・運行日を表示

#### 直通運転表示
- `connections.csv` に接続が定義された列車は、時刻表末尾に**緑色の「直通先」行**を追加
- 接続チェーンを辿り最終目的地の駅名・到着時刻を表示
- クリックで接続先路線の時刻表に遷移

---

### ダイヤグラムページ（diagram.html）

#### 表示
- X軸: 時刻、Y軸: 駅（実距離比 or 等間隔）
- 下り列車: 青系色、上り列車: 赤/橙系色
- 列車ごとに異なる色を自動付与

#### 操作
| 操作 | 動作 |
|------|------|
| ドラッグ | 移動（パン） |
| スクロール | 時間軸・駅間を同時ズーム |
| Ctrl+スクロール | 時間軸のみズーム |
| Shift+スクロール | 駅間のみズーム |
| ↔ ± ボタン | 時間軸ズーム |
| ↕ ± ボタン | 駅間ズーム |
| ⟳ ボタン | ビューリセット |

#### フィルタ
- 路線・方向（上り/下り/上下同時）・ダイヤ種別・日付指定

#### 描画方式
- オフスクリーンキャンバスにキャッシュ → パン時は高速ビットブリット
- ズーム・スクロール停止後にデバウンスで高品質再描画（ベクター）
- テキスト（駅名・時刻・列車番号）はズームに依存しない固定サイズ

---

## 制約・注意事項

- 1つの路線のすべての列車データは `data/{路線名}.csv` の1ファイルに記述する
- 列車番号は同一路線・同一運行日の組み合わせで一意である必要がある（同一番号で平日・休日を別定義する場合は運行日で区別）
- 時刻は **HHMM整数形式**（例: `730` = 7:30, `1245` = 12:45）
- 日付を跨ぐ列車（深夜便）は `2400` 以降の表記を使用（例: `2530` = 翌1:30）
- `connections.csv` が存在しない場合は直通機能は無効（エラーにはならない）
- ブラウザから直接ファイルを開くと CORS エラーになるため、必ず `serve.bat` 経由で使用する
